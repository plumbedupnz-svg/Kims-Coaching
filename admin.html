Email Settings

All booking, shop, waitlist, and admin emails use Resend. You can update the public sender name, from email, and reply-to email here. The Resend API key is managed securely in Vercel Environment Variables.
Advanced setup notes

Vercel requires: EMAIL_PROVIDER=resend, RESEND_API_KEY, EMAIL_FROM_NAME, EMAIL_FROM_ADDRESS, EMAIL_REPLY_TO, and optionally EMAIL_ADMIN_TO. EMAIL_FROM_ADDRESS must use a domain verified in Resend.

Account confirmations and password resets: these are sent by Supabase Auth. Connect Resend in Supabase Dashboard under Authentication > Email > SMTP Settings using smtp.resend.com, port 465, username resend, and a Resend API key as the password.
